    WOG is short for World of Greenwood, a free, open source game.
    WOG is a RPG game on Android mobil phone, one of auther email: Lional.King@gmail.com
    WOG source code is on github, web site:  http://github.com/wzl7222504/WOG
    Any question about WOG are welcom to communicate on web site:  http://netsurfers.gicp.net/bbs
    Support port and land screen with any size.
    Save data with custom sqlite which extends class provider.
    Draw graphics with class surface view and respond to user operation.
